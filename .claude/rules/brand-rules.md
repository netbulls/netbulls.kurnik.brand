# Brand Rules

## The name
Kurnik = Polish for "chicken coop". This is an inside joke. The mark does NOT illustrate this.

## Mark constraints (strict)
- **Monochrome only**: `#d4840a` (amber) on `#0e0c0a` (near-black). No multi-color. No opacity tricks for color variation.
- **Abstract**: NOT chicken, rooster, egg, comb, feather. Not illustrative.
- **Scalable**: Must read cleanly at 32px. No detail that disappears small.
- **Vary via**: stroke weight, fill vs outline, positive/negative space, geometry — not color
- **Sensibility**: Figma, Linear, Vercel. Infrastructure-feel. Confident, minimal.

## Color palette
| Token | Hex | Usage |
