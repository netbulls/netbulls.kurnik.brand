# Kurnik Brand Identity

> Where ideas come home to roost 🐔

Brand identity assets and documentation for [Kurnik](https://kurnik.ai) — an AI-powered product incubator.

## Structure

```
site/               Brand documentation (deployable)
  brand-foundation.html   Phase 1: Brand Foundation (positioning, personality, audience)
phase-2/            Symbol exploration (upcoming)
phase-3/            Ranking & evaluation (upcoming)
brand/              Final brand assets (upcoming)
  logo/
  icon/
  colors/
  typography/
```

## Brand Summary

- **Name:** Kurnik (Polish for "chicken coop")
- **Domain:** kurnik.ai
- **Positioning:** Product incubator — messy ideas in, coherent product vision out
- **Personality:** Warm & confident (Notion meets Stripe)
- **Symbol:** Abstract rooster — strong presence, stylized, geometric
- **Colors:** Dark & rich — dark backgrounds, warm amber accents
- **Audience:** Founders & aspiring entrepreneurs (day one), dev teams (later)

## Design Process

1. \u2705 Brand Foundation — positioning, audience, personality, evaluation framework
2. \u2B1C Symbol Exploration — 15-20 concepts across 4 territories
3. \u2B1C Scientific Ranking — 100-point evaluation
4. \u2B1C Refinement — top 3 deep dive
5. \u2B1C Logo Development — variants, colors, responsive
6. \u2B1C Brand Book — final identity package
