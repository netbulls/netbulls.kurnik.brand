# DECISIONS.md

Full decision history for **netbulls.kurnik.brand** — the *why* behind brand choices. Written for Claude Code continuity. Read this before starting any brand session.

---

## 2026-02-21 · Warsaw, PL — Bootstrap session

### Name selection
Explored: Prism (too crowded, OpenAI conflict), Sift (taken), Glean ($7B company conflict), Kura (multiple conflicts).

**Final: Kurnik** — Polish for "chicken coop." Domain kurnik.ai registered. Inside joke — PlayOK (nostalgic Polish games site) is kurnik.pl. Memorable, warm, nobody in tech has it.

**Critical implication: the mark does NOT illustrate the name.** Chicken/rooster/egg/comb references explored in Phase 02 Round 01 and abandoned. The name is the joke. The mark stands alone.

### Audience
Primary: founders, aspiring entrepreneurs, people with capital but no methodology. NOT dev teams initially. Affects tone — approachable and credible, not technical.

### Personality
Warm & confident. Notion meets Stripe. Premium incubator feel. Not playful, not clinical.

### Color rationale
- Dark = premium, focused, serious (Linear, Vercel, Raycast)
- Amber = warmth, approachability — breaks the cold tech blue/purple cliché
- Stone scale for neutrals — warm undertone, not cold

Final tokens:
- `#0e0c0a` — near-black, warm undertone
- `#d4840a` — amber primary / mark color
- `#f5a623` — amber highlight
- `#e8ddd0` — warm off-white text
- `#2a2420` — border/divider

### Typography
DM Serif Display (display) + Manrope (UI) + DM Mono (code/labels). Warmth, distinctiveness, readability at small sizes.

## Phase 02 — Symbol exploration

### Round 01 (inconclusive)
- 90 SVG concepts across 4 territories: Geometric Rooster, Egg+Rooster Fusion, Comb/Crown, Nested/Negative Space
- None compelling. Territories too literal — forced chicken/rooster references.
- Lesson: don't illustrate the name.
- Semi-promising:
  - **A20** Circuit Rooster — systematic, constructed, engineered quality
  - **B13** Half Shell — arch + legs, architectural feel
  - **B10** Emergence — shape rising from base, upward energy
- Thread: **structured geometry with upward/emergent energy**

### Technical lesson from Round 01
Large HTML files (>50kb) must be pushed via local git. GitHub API truncates large content fields. All future rounds must be generated locally and pushed via `git push`.

### Monochrome rule — why
Established after Round 01 review. Opacity variations were creating false color variety, masking weak shapes. Strict monochrome forces the mark to work on geometry alone — which is the real test of a good logo.

### Round 02 direction
Expanded to 9 abstract territories (added Phase Shift, Stack/Depth, Symbiosis; dropped Letterform K):
1. **A: Upward / Emergence** — shapes rising from a base
2. **B: Contained / Portal** — negative space as subject
3. **C: Systematic / Grid** — modular, engineered, precise
4. **D: Radial / Convergence** — elements from/toward center
5. **E: Tension / Duality** — two shapes in dialogue
6. **F: Threshold / Phase Shift** — state change, transition moments
7. **G: Stack / Depth** — layered planes, dimensional depth
8. **H: Precision / Calibration** — crosshairs, targets, alignment
9. **I: Interface / Symbiosis** — connection, integration points

### Round 02 results (270 concepts → 17 shortlisted)

**Leading direction: Target / Calibration (Territory H)**
The crosshair vocabulary maps directly to what Kurnik does — keep projects focused, on target. "Most common mistake in projects = lack of focus, lack of priorities. People drift away. Kurnik helps them win, hit the target."

User prefers **bold and defined** marks over subtle/delicate. "It needs to be simple. Target is simple."

**Shortlist (17 picks):** H29, H02, H10, H13, H15, H20, B05, B03, B10, B25, C22, D18, D09, E26, F06, F20, G12

**Territory evaluation:**
- **H (Calibration):** Strongest. 6 picks. Semantic match to product = focus.
- **B (Portal):** 4 picks. Progress/incompleteness narrative (B03, B10). Simple void (B05).
- **D (Convergence):** 2 picks. Groups with H — more target vocabulary.
- **F (Phase Shift):** 2 picks. Chevron cascade (F20) = iterative progress. Split diamond (F06) = potential vs realized.
- **E (Duality):** 1 pick. Convergent triangles (E26) = productive tension.
- **G (Stack):** 1 pick. Isometric planes (G12) = "foundations matter."
- **C (Grid):** 1 pick. Bridge span (C22) = "bridging the gap."
- **A (Emergence):** 0 picks. Didn't resonate.
- **I (Symbiosis):** 0 picks. Didn't resonate.

**Conceptual reasoning per pick:**
- B03/B10: "Something in progress but not finished." Incubator = always building.
- B05: Dead simple. Void in middle is intriguing but unclear how to develop.
- B25: Direction concept = YES, but needs proper solid arrow (road-sign style, not twisted).
- C22: "Bridging the gap." More to do with the squares concept.
- E26: Opposites pushing against each other. Tension = productive force.
- F06: Diamond empty one way, full the other. Duality of state.
- F20: "Very strong direction through iterations." Each chevron = a step toward the goal.
- G12: "Many layers. What you see on the surface... foundations matter."

**Complexity ceiling:** F20 (4 chevrons) and G12 (3 planes) = upper limit. More = noisy.

**Dropped from shortlist for Round 03:**
- H29 (mil-dot): dots could symbolize tasks but "that's overcomplicating it"
- H10 (floating dots): same issue — too busy

### Emerging brand themes
1. **Focus / Target** — primary. H-series, D-series.
2. **Progress / Iteration** — F20 chevrons, B03/B10 incomplete rings
3. **Direction / Momentum** — B25 arrow, F20 chevrons
4. **Duality / Transformation** — F06, E26
5. **Foundation / Depth** — G12 layers
6. **Bridging / Connection** — C22

Key insight: user thinks in brand narratives, not just aesthetics. Each shape carries a story about what Kurnik does.

### Sensibility reference
Figma, Linear, Vercel, Stripe. Infrastructure-feel. Confident, minimal, slightly cold. Not playful, not illustrative.

### Evaluation framework (100 points)
- Visual Design: 30pt
- Symbolic Meaning: 25pt
- Color Strategy: 20pt
- Typography-Icon Harmony: 15pt
- Audience Resonance: 10pt

### Round 03 (60 concepts — diminishing returns)
- 60 SVGs across 5 refined tracks based on R02 shortlist feedback
- User verdict: "from this new round there is nothing that would feel substantially more appropriate than from the previous one"
- Key feedback: "feels like there are too many constraints to you" → decided to drop analytical framework
- Lesson: over-constraining iteration kills serendipity. Looser briefs produce better results.

### Round 04 (100 concepts — fresh territories, loose briefs)
- 100 SVGs across 10 completely fresh territories: Slash, Lock, Impossible, Stamp, Knot, Signal, Totem, Fragment, Orbit, Glyph
- Generated with loose creative briefs, no starting points, no design direction panels
- More variety emerged, but no breakthrough. Confirmed existing shortlist was strong.

### Iteration Review — deep feedback on 11 concepts

User rated and commented all 11 surviving concepts. Key takeaways:

**Strong contenders (rating 4):**
- **B25 Arrow Void** — loves the surrounding squircle shape. Wants arrow variations. Simplest: right-turn arrow (up then right).
- **F20 Chevron Cascade** — likes it a lot. Direction should point "into success" — upward. Strong contender.
- **E26 Convergent Triangles** — solid, shows tension and pushing against each other.
- **F06 Split Diamond** — very nice. Experiment with weight variations.
- **G12 Isometric Planes** — likes it but worried it's too common (needs uniqueness research).
- **T5-02 Stacked Slabs** — crazy simple = interesting. But must show building up/growing — strong base, more detail toward top.
- **T1-10 Concentric Rotated Squares** — "asking to be bolder, way bolder. Like winning or something."

**Recurring themes:**
1. Direction/momentum matters — arrows should point "into success" (up or right)
2. Weight hierarchy should match meaning — strong foundation, building upward
3. Boldness requested repeatedly — "way bolder, like winning"
4. Concern about originality (G12 isometric planes too common)
5. Wants to see concepts alongside "kurnik" text

- Deep iteration on each of the 11 surviving concepts
